get

Récupère la liste des contraventions entre deux dates.

get /contrevenants

Récupère la liste des contraventions entre deux dates.

  • Request
  • Response

Query Parameters

  • du: required(string)

    Date de début (ISO 8601, ex. 2022-05-08).

    Example:

    2022-05-08
  • au: required(string)

    Date de fin (ISO 8601, ex. 2024-05-15).

    Example:

    2024-05-15

/etablissements

/etablissements get

get

Récupère la liste de tous les établissements ayant au moins une infraction, triée par ordre décroissant du nombre total d'infractions connues pour chaque établissement.

get /etablissements

Récupère la liste de tous les établissements ayant au moins une infraction, triée par ordre décroissant du nombre total d'infractions connues pour chaque établissement.

  • Response

HTTP status code 200

Succès - Retourne un tableau JSON des établissements et leur compte d'infractions.

Body

Media type: application/json

Type: array of object

Items: items

  • etablissement: required(string)

    Le nom de l'établissement.

  • nombre_infractions: required(integer)

    Le nombre total d'infractions enregistrées pour cet établissement.

Example:

[
  {
    "etablissement": "RESTAURANT BASHA",
    "nombre_infractions": 40
  },
  {
    "etablissement": "DOUBLE PIZZA",
    "nombre_infractions": 34
  },
  {
    "etablissement": "BUFFALO BILL WINGS",
    "nombre_infractions": 30
  },
  { ... }
]
